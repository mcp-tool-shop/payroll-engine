# Roadmap (Build-First)

## Phase 2A — Financial Spine
- PSP ledger tables + posting invariants
- Funding requests + inbound settlement events
- Safe-to-commit funding gate
- Employee payment orchestration (ACH credit) + reconciliation loop

## Phase 2B — Obligations
- Tax liability generation from payroll line items
- Third-party obligations (401k/HSA/garnishment) generation + payments
- Garnishment priority + arrears ledger integration

## Phase 2C — Compliance Ops
- Filing job framework (draft → submitted → accepted/rejected)
- Notice management intake + linkage
- Amendment generation via deltas

## Phase 2D — Risk & Controls
- KYB/KYC hooks
- Admin change risk scoring + cooling-off
- Velocity limits and anomaly detection
